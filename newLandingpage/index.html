  • • Overpaying for the same item

Order vs Received Analysis

What this screen does

Compares purchase order data with received stock. Matches items using name and pack size. Automatically highlights mismatches.

What problems it catches

  • • Items received but not ordered
  • • Items ordered but not received
  • • Quantity mismatches
  • • Pack size differences
  • • Extra or missing stock entries

Why this matters

Helps pharmacy owners catch supplier discrepancies before accepting stock or finalizing payments.
