# Color System Specification - Landify-Inspired Design

## Global Color Tokens

### Primary Palette (Muted Purple-Based)
- **Primary Purple**: `#6B5CE6` - Main brand color, used for headings and primary elements
- **Primary Purple Dark**: `#5A4FCF` - Hover states, emphasis
- **Primary Purple Light**: `#8B7EE8` - Lighter accents, secondary elements
- **Primary Purple Muted**: `#E8E5F5` - Subtle backgrounds, borders

### Secondary Palette (Supporting Purples)
- **Secondary Purple**: `#7C3AED` - Complementary purple for variety
- **Secondary Purple Light**: `#A78BFA` - Lighter secondary accents
- **Secondary Purple Muted**: `#F3F0FF` - Very light backgrounds

- **Warm Accent**: `#F59E0B` - Primary CTA buttons, highlights, key actions
- **Warm Accent Dark**: `#D97706` - CTA hover states
- **Warm Accent Light**: `#FCD34D` - Light accents, badges
- **Warm Accent Muted**: `#FEF3C7` - Subtle warm backgrounds

- **Neutral White**: `#FFFFFF` - Primary backgrounds
- **Neutral Gray 50**: `#F9FAFB` - Light section backgrounds
- **Neutral Gray 100**: `#F3F4F6` - Card backgrounds, subtle sections
- **Neutral Gray 200**: `#E5E7EB` - Borders, dividers
- **Neutral Gray 400**: `#9CA3AF` - Secondary text, placeholders
- **Neutral Gray 600**: `#4B5563` - Body text, secondary content

- **Text Secondary**: `#4B5563` - Body text, descriptions

## CTA Color Rules

### Primary CTA (Hero, Final CTA)
- **Background**: `#F59E0B` (Warm Accent) - Warm, attention-grabbing
- **Text**: `#FFFFFF` (Text Inverse) or `#111827` (Text Primary) for contrast
- **Hover**: `#D97706` (Warm Accent Dark)
- **Usage**: Main conversion points, hero section, final CTA section

### Secondary CTA (Solution, Features)
- **Background**: `#6B5CE6` (Primary Purple) - Brand-aligned
- **Text**: `#FFFFFF` (Text Inverse)
- **Hover**: `#5A4FCF` (Primary Purple Dark)
- **Usage**: Supporting CTAs, feature cards, solution section

### Tertiary CTA (Modal Submit)
- **Background**: Gradient from `#6B5CE6` to `#7C3AED` - Premium feel
- **Text**: `#FFFFFF` (Text Inverse)
- **Hover**: Darker gradient variant
- **Usage**: Form submissions, modal actions

## Gradient Specifications

### Primary Gradient (Final CTA, Modal Submit)
- **Direction**: 135deg (diagonal)
- **Colors**: `#6B5CE6` → `#7C3AED`
- **Usage**: High-impact sections, premium CTAs

### Accent Gradient (Subtle Backgrounds)
- **Direction**: 135deg (diagonal)
- **Colors**: `rgba(107, 92, 230, 0.05)` → `rgba(124, 58, 237, 0.05)`
- **Usage**: Hero section, subtle accents

### Feature Accent Bar Gradient
- **Direction**: Vertical (to bottom)
- **Colors**: `#6B5CE6` → `#7C3AED`
- **Usage**: Primary feature card accent bar

## Common Mistakes to Avoid

### ❌ DON'T:
1. **Mix warm and cool accents randomly** - Use warm accent (`#F59E0B`) only for primary CTAs, purple for everything else
2. **Use different purple shades inconsistently** - Stick to the defined primary/secondary purple roles
3. **Vary text colors within same hierarchy** - All H2s should be `#6B5CE6`, all body text `#4B5563`
4. **Create section-specific color schemes** - Maintain consistency across sections
5. **Use gradients without purpose** - Only use gradients for final CTA and modal submit
6. **Overuse the warm accent** - Reserve it for primary conversion points
7. **Mix background patterns** - Alternate between `#FFFFFF` and `#F9FAFB` consistently
8. **Vary border colors** - Use `#E5E7EB` for all standard borders
9. **Change CTA button styles** - Maintain rounded-full for primary, rounded-lg for secondary
10. **Use random opacity values** - Standardize opacity (0.05 for subtle, 0.9-0.95 for text on gradients)

### ✅ DO:
1. **Maintain purple as primary brand color** - Use `#6B5CE6` for all headings and brand elements
2. **Reserve warm accent for primary CTAs** - Creates clear visual hierarchy
3. **Use consistent neutral backgrounds** - Alternate white and light gray systematically
4. **Apply high contrast for accessibility** - Text on backgrounds meets WCAG AA standards
5. **Reuse color tokens** - Reference CSS variables, don't hardcode hex values
6. **Maintain consistent spacing** - Colors should enhance, not distract from structure
7. **Use subtle gradients sparingly** - Only for high-impact sections
8. **Keep borders subtle** - Light gray borders maintain clean aesthetic
9. **Standardize text hierarchy** - Primary/Secondary/Tertiary text colors are consistent
10. **Test contrast ratios** - Ensure all text meets accessibility standards

## Accessibility Notes

- **Text on White (`#111827` on `#FFFFFF`)**: Contrast ratio 16.6:1 ✅ AAA
- **Text on Light Gray (`#4B5563` on `#F9FAFB`)**: Contrast ratio 7.2:1 ✅ AA
- **Text on Purple (`#FFFFFF` on `#6B5CE6`)**: Contrast ratio 4.5:1 ✅ AA
- **Text on Warm Accent (`#FFFFFF` on `#F59E0B`)**: Contrast ratio 2.8:1 ⚠️ (Use `#111827` for better contrast)
- **Text on Gradient**: Ensure sufficient contrast, may need to adjust opacity

## Implementation Priority

1. **Update CSS variables** - Define all color tokens in `:root`
2. **Apply to headings** - Replace all H2 colors with primary purple
3. **Update backgrounds** - Apply alternating white/gray pattern
4. **Standardize CTAs** - Apply warm accent to primary, purple to secondary
5. **Update cards** - Apply consistent border and background colors
6. **Refine gradients** - Update gradient definitions
7. **Test accessibility** - Verify contrast ratios
8. **Polish details** - Bullets, badges, icons, dividers
